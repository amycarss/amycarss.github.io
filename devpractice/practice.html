VSCode Shortcuts

To get the html doctype in VS code, create a html file and save it, then go into your file in VS code and type an exclamation point. While you're holding the keys down, the doctype will appear in a little modal window, and hitting enter will paste it into your file.

Yay!

And to get Live Server Preview to work, go to the html file, press F1 and select "Show Live Server Preview" from the dropdown.
